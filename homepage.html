HTML - это язык гипертекстовой разметки текста. Он нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы и видео

JavaScript - это язык, который позволяет вам применять сложные вещи на web странице — каждый раз, когда на web странице происходит что-то большее, чем просто её статичное отображение

CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки

React — JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов
